The extension will save your highlights locally in your browser, so you can access them later by clicking the extension icon. Each highlight includes the text you selected, the page title, URL, and timestamp.

Features

Smart Detection: The extension automatically detects when you select text and positions the save popup intelligently near your selection.

Local Storage: All your highlights are stored locally in your browser for complete privacy. No data is sent to external servers.

Easy Management: View, search, copy, and delete your highlights through the extension popup. You can also navigate back to the original pages where you made the highlights.

Modern UI: Beautiful, responsive design with smooth animations and intuitive controls that make managing your highlights a pleasure.

Technical Details

This extension is built with modern web technologies including React, TypeScript, and Tailwind CSS. It uses the Chrome Extension Storage API to persist your highlights locally and provides a seamless user experience across all websites.

The content script runs on all web pages and detects text selections in real-time. When you select text, it creates a floating popup that allows you to save the highlight with a single click.
